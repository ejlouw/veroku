special_evidence – (dict) evidence dictionary (mapping variable names to values) that should be used in the calculation of messages, and not to reduce factors. This allows factor approximations - such as the non-linear Gaussian to be iteratively refined.

process_graph(tol=0.001, max_iter=50)[source]¶

Perform synchronous message passing until convergence (or maximum iterations). :param tol: The minimum tolerance value for the KL divergence D_KL(previous_message || next_message) that needs

to be reached (for all messages) before stopping message passing (before max_iter is reached).

Parameters

max_iter – The maximum number of iterations of message passing. The maximum number of messages that can be passed is max_iter * n, where n is the number of message paths (2x the number of edges) in the graph.
