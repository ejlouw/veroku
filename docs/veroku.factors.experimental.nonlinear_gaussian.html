class NonLinearGaussian(conditioning_vars, conditional_vars, transform, noise_cov, log_weight=0.0, joint_distribution=None, conditional_update_factor=None, conditioning_factor=None, observed_evidence=None)[source]¶

Bases: veroku.factors._factor.Factor

A Class for instantiating and performing operations on multivariate Gaussian mixture functions.

The initializer.

Parameters

  • conditioning_vars (list) – The conditioning vars.

  • conditional_vars (list) – The conditioning vars.

  • transform – The transformation that links the conditioning to the conditional distribution

  • noise_cov – The noise covariance matrix of the zero mean Gaussian noise.

  • conditional_update_factor (Gaussian) – A factor, with same scope, subset of, the conditional vars, to be multiplied with the transformed (joint) distribution to form the final joint distribution.

  • conditioning_factor (Gaussian) – The factor with scope less than or equals to the conditional variables to be transformed by the transform. If the scope is less than the conditional variables set, observed evidence (if available) can be used to still enable the successful transformation and calculation of the joint.

  • observed_evidence (dict) – A dictionary mapping variable names ot observed values.

class NonLinearGaussianTemplate(conditioning_var_templates, conditional_var_templates, transition_function, noise_cov)[source]¶

Bases: veroku.factors._factor_template.FactorTemplate

A template class for NonLinearGaussian factors.

The initializer.

Parameters

  • conditioning_var_templates (str list) – The list of formattable strings for the conditioning variables (i.e: [‘var_a{i}_{t}’, ‘var_b{i}_{t}’])

  • conditional_var_templates (str list) – The list of formattable strings for the conditional variables (i.e: [‘var_c{i}_{t}’, ‘var_d{i}_{t}’])

  • conditioning_var_templates – The formattable strings for the conditioning variables.

  • conditional_var_templates – The formattable strings for the conditioning variables.

  • transition_function (callable) – The function that specifies the non-linear transform. This function takes 2 parameters, a vector-like value to be transformed and the variable names list specifying the names of the variable elements of the value vector (i.e transition_function = lamda x, var_names: np.square(x)). The variable names do not need to be used, but can be useful in certain cases where functions need to be applied to specific variables.

  • noise_cov – The noise covariance matrix of the additive noise random variable.
