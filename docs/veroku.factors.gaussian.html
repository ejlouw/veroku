veroku.factors.gaussian module¶

class Gaussian(cov=None, mean=None, log_weight=None, prec=None, h_vec=None, g_val=None, var_names=None)[source]¶

Bases: veroku.factors._factor.Factor

A class for instantiating and performing operations om multivariate Gaussian distributions.

General constructor that can use either covariance form or canonical form parameters to construct a d-dimensional multivariate Gaussian object.

Parameters

  • cov (dxd numpy.ndarray or float) – The covariance matrix (or variance scalar in 1-dimensional case).

  • mean (dx1 numpy.ndarray or float) – The mean vector (or mean scalar in 1-dimensional case).

  • log_weight (float) – the log of the weight (the Gaussian function integrates to the weight value)

  • prec (dxd numpy.ndarray or float) – the precision matrix (or precision scalar in 1-dimensional case).

  • h_vec (dx1 numpy.ndarray or float) – the h vector (or h scalar in 1-dimensional case).

  • g_val (float) – the g parameter

  • var_names (str list) – a list of variable names where the order of the names correspond to the order in the mean/covariance or K/h parameters
