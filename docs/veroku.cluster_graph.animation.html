veroku.cluster_graph.animation.change_cluster_graph_edge_color(graph, node_a_name, node_b_name, new_color)[source]¶

Change the color of an edge in a cluster graph graphviz object. Note that the cluster graph has cluster and sepset nodes. The ‘edge’ of which the color is changed in this function is actually two edges in the graphviz object (the edge between node_a and the sepset node and the edge between the sepset node and node_b). :param graph: The graphviz graph :param node_a_name: :param node_b_name: :param new_color: :return:

Check that both sub-strings can be found in seperate locations in strings Examples:
